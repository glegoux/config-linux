# Jupyter

http://jupyter.org/

# Install

~~~
pip install jupyter
~~~

# Extension

https://github.com/ipython-contrib/jupyter_contrib_nbextensions

~~~
pip install https://github.com/ipython-contrib/jupyter_contrib_nbextensions/tarball/master
~~~

In the `~/.jupyter/` folder.
